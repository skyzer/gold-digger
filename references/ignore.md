# Ignore list

Tickers and project slugs Gold Digger should never track, surface, or auto-scout.
Edit freely — add comments with `#` to explain why something is ignored.

The matcher is **case-insensitive** and matches:
- Any line starting with `- ` is treated as an entry
- Trailing text after `#` is a comment (ignored)
- Both tickers (e.g. `BTC`) and project slugs (e.g. `bitcoin`) are checked
